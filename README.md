# Goods Delivery
[![Last Release:](https://img.shields.io/github/release/openconnect/openconnect-gui.svg)](https://github.com/iamtienng/Goods-Delivery/releases)

### Pre-requisites
* git - [Installation guide](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git).
* node.js - [Download page](https://nodejs.org/en/download/).
* npm - comes with node or download yarn - [Download page](https://yarnpkg.com/lang/en/docs/install).
* mongodb - [Download page](https://www.mongodb.com/download-center/community).

### Installation 
``` 
git clone https://github.com/iamtienng/Goods-Delivery.git
``` 
Open another first bash for back-end:
``` 
$ cd Express && npm i
$ npm start
``` 
Open another second bash for front-end:
``` 
$ cd Angular && npm i
$ ng serve --open
```
